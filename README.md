# SECTOR ZERO 

A browser-based raycasting FPS game inspired by classic shooters like Doom and Wolfenstein. Built entirely with vanilla JavaScript and HTML5 Canvas — no game engine, no libraries, just raw code.

---

## 🎮 Play it
After Installing the file 
Open `SECTOR-ZERO.html` in any browser.

---

## Features

- Raycasting 3D rendering engine built from scratch
- Enemy AI that tracks and chases the player
- Mobile touch controls — fully playable on phone
- Weapon system with shooting mechanics
- Collision detection
- Minimap for navigation
- Performance optimized for smooth gameplay

---

## Controls

**Desktop**
| Key | Action |
|-----|--------|
| W / ↑ | Move Forward |
| S / ↓ | Move Backward |
| A / ← | Turn Left |
| D / → | Turn Right |
| Space | Shoot |

**Mobile**
| Control | Action |
|---------|--------|
| Left joystick | Move |
| Right joystick | Look / Turn |
| Fire button | Shoot |

## Tech Stack

- **Language:** JavaScript (Vanilla)
- **Rendering:** HTML5 Canvas API
- **Engine:** Custom raycasting renderer
- **Platform:** Browser (Desktop + Mobile)

---

## How It Works

SECTOR ZERO uses a raycasting algorithm to simulate 3D perspective from a 2D map — the same technique used in classic 90s shooters. Rays are cast from the player's viewpoint, and walls are drawn taller or shorter depending on how far the ray travels before hitting a wall.
